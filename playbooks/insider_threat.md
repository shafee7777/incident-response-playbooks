## Identification
- Unusual file downloads or data transfers
- Accessing systems outside of normal working hours
- Privilege misuse or unauthorized access attempts
- Alerts from UEBA or insider threat tools

## Containment
- Restrict user access permissions immediately
- Disable user accounts involved
- Preserve logs for legal/investigative review
- Monitor affected systems for ongoing activity

## Eradication
- Remove unauthorized tools or data exfiltration utilities
- Terminate malicious processes
- Tighten access controls and least-privilege policies

## Recovery
- Restore necessary system access with proper controls
- Conduct forensic review of system changes
- Enhance monitoring and anomaly detection

## Lessons Learned
- Review HR/IT policies for insider threat prevention
- Improve access control and certification reviews
- Provide enhanced security awareness and behavioral training
